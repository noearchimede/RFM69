Descrizione dettagliata

Permette di assegnare un valore del tipo corretto a una qualsiasi variabile aritmetica.

I tipi accettati sono:

  • bool
  • uint8_t, uint16_t, uint32_t
  • int8_t, int16_t, int32_t
  • float, double
