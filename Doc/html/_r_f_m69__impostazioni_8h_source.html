18 // Costanti di impostazione di tutti i registri della radio RFM69
19 //
20 // Ogni costante è presentata da due linee di commento: la prima è una descrizione
21 // dell'impostazione con l'indirizzo del registro della radio in cui è
22 // registrata, la seconda contiene un'indicazione sul tipo di valore da
23 // inserire e, nel caso di impostazioni "a scelta multipla", tutti i valori
24 // possibili.
25 //
26 // La notazione `_VALORE1,, _VALORE2, ecc` per un'impostazione `IMPOSTAZIONE`
27 // significa che il suo valore deve essere una delle costanti
28 // `IMPOSTAZIONE_VALORE1`, `IMPOSTAZIONE_VALORE2`, ecc.
29 //
30 // `ON OFF` Significa che il valore può essere la costante `ON` o `OFF`.
31 //
32 // `x` significa che il valore richiesto è un numero che sarà scritto direttamente
33 // nel registro corrispondente.
34 //
35 // `x [auto]` significa che il valore è un numero che sarà convertito automaticamente
36 // (da una macro del preprocessore) nel valore da inscrivere nel registro.
