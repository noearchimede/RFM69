Inizializza la radio. Deve essere chiamato all\textquotesingle{}inizio del programma. 

La funzione esegue le seguenti operazioni in questo ordine\+:
\begin{DoxyItemize}
\item controllo che la classe non debba gestire troppe radio (cioé più di una)
\item prepara l\textquotesingle{}interfaccia S\+PI per comunicare con la radio
\item (eventualmente esegue il reset della radio)
\item controlla che un dispositivo sia connesso
\item controlla che il dispositivo sia una radio R\+F\+M69
\item collega l\textquotesingle{}interrupt della radio all\textquotesingle{}I\+SR della classe
\item scrive tutti i registri della radio inserendovi le impostazioni stabilite nel file \doxyref{R\+F\+M69\+\_\+impostazioni.\+h}{pag.}{_r_f_m69__impostazioni_8h}
\item inizializza alcune variabili della classe
\item crea un buffer di {\ttfamily lunghezza\+Max\+Messaggio} bytes che resterà allocato fino alla distruzione dell\textquotesingle{}istanza della classe.
\end{DoxyItemize}

\begin{DoxyNote}{Nota}
L\textquotesingle{}esecuzione di questa funzione richiede alcuni decimi di secondo.

Sarà allocata un\textquotesingle{}array di {\ttfamily lunghezza\+Max\+Messaggio} bytes.
\end{DoxyNote}

Invia un messaggio ripetutamente fino alla ricezione di un A\+CK. 

Questa funzione invia un messaggio, aspetta l\textquotesingle{}A\+CK e se non lo riceve entro il tesmpo specificato in {\ttfamily \doxyref{imposta\+Timeout\+Ack()}{pag.}{class_r_f_m69_a440d37e3362fc5249c62a88eb2d472d1}} lo invia di nuovo. Ripete questa operazione per al massimo {\ttfamily tentativi} volte, dopo le quali restituirà 1 ({\ttfamily Errore\+::\+Lista\+Errori\+::errore}) se il messaggio non è ancora stato confermato (quindi probabilmente non è arrivato). Restituirà invece 0 ({\ttfamily Errore\+::\+Lista\+Errori\+::ok})subito dopo aver ricvevuto un A\+CK.

Stampa la descrizione di un errore sul monitor seriale. 

Questa funzione permette di stampare sul monitor seriale la causa di un errore segnalato da una funzione di questa classe.

Funziona per i codici di errore definiti dall\textquotesingle{}{\ttfamily enum} \doxyref{Errore\+::\+Lista\+Errori}{pag.}{struct_r_f_m69_1_1_errore_a4dadb6af0e4d38eaac18d3afe26e5bcd}.

\begin{DoxyNote}{Nota}
Le stringhe di testo sono salvate nella memoria flash (del programma), non nella S\+R\+AM (delle variabili), e pesano circa 0.\+5kB.
